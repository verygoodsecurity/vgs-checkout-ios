VGSCheckoutUnknownPaymentCardModel

public struct VGSCheckoutUnknownPaymentCardModel

An object representing Unknown Payment Cards - cards not defined in the SDK. Object is used when validation forCardBrand.unknown is set as true. Check VGSValidationRulePaymentCard for more details. Validation attributes can be edited through `VGSCheckoutPaymentCards.unknown model.
